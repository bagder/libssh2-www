libssh2 www
===========

This is the contents of the web site for libssh2, libssh2.org.

Requirements
============

* GNU make
* fcpp - https://github.com/bagder/fcpp
* roffit - https://github.com/bagder/roffit

Steps
=====
1. ln -s [libssh2 git repo]/docs .

2. ln -s [libssh2 git repo] git-source

3. make
